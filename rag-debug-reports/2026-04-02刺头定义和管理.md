- **性能瓶颈**:
  - 🔴 **LLM API 调用**: 占总耗时的 96.6%
  - 🟢 **Embedding 生成**: 4.23s 可接受（可优化批处理）
  - 🟢 **向量搜索**: 0.06s 非常快，符合预期

- **优化建议**:
  - [性能] 检查 glm-4.7 API 响应时间，考虑切换到更快的模型或使用流式响应优化首字延迟
